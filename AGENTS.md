# AGENTS.md — Bot Zalo

> **Rule #1 — Env**: Every time you start coding, run the env check below. If anything is missing, **install it** before editing any code.
> **Rule #2 — Auto-push**: After any task where you edited code (and validation passed), **commit + push to `origin main` immediately**. No waiting for user approval.

---

## Rule #1 — Pre-Code Env Check

```bash
# 1. Bun runtime ← REQUIRED (bot runs on Bun, tests use bun:test)
which bun || { echo "❌ BUN MISSING"; npm install -g bun; }

# 2. TypeScript checker
cd apps/bot && (npx --no-install tsc --version 2>&1 || \
  echo "❌ TSC MISSING → cd apps/bot && npm install --save-dev typescript")

# 3. Node (fallback runtime, ≥ 18)
node --version

# 4. Gemini SDK present
ls apps/bot/node_modules/@google/genai/dist/genai.d.ts && echo "✓ SDK present"

# 5. Sanity smoke: typecheck the project
cd apps/bot && npx tsc --noEmit 2>&1 | tail -5
```

If any `❌` block printed → **run the install command it suggests**, then re-run the check until clean.

### Install Cheatsheet

| Missing | Install |
|---|---|
| `bun` | `npm install -g bun` |
| `tsc` | `cd apps/bot && npm install --save-dev typescript` |
| `biome` | `cd apps/bot && npm install --save-dev @biomejs/biome` |
| Test deps | `bun install` (auto-runs from `bun.lock`) |

## Rule #2 — Auto-push to origin main

After completing any task that edited source files AND passed validation (typecheck / lint / tests), **commit + push immediately**:

```bash
cd /workspaces/botzalo

# Sanity-check git identity (must be set, NEVER fake it)
git config user.email >/dev/null || { echo "❌ git user.email not set → STOP, ask user"; exit 1; }
git config user.name  >/dev/null || { echo "❌ git user.name  not set → STOP, ask user"; exit 1; }

git pull --rebase origin main 2>/dev/null || true   # safe if no remote commits
git status --short
git add <changed-files>      # explicit, never .env / secrets / bun.lock
git commit -m "<auto-generated summary>"
git push origin main
```

### Safety rules

- **NO force-push**, **NO push --force-with-lease**, **NO interactive rebase**
- **NO fake identity** — if `user.email`/`user.name` not set, STOP and ask user. **Never** default to bot credentials.
- Skip ONLY if user explicitly says "don't push" or "stay local"
- Never push `.env`, `secrets.*`, `bun.lock` (also excluded from `git add`)
- Never commit half-done work (commit only after the task is *truly* done + verified)
- If `git push origin main` fails with non-fast-forward → STOP and ask user; do **not** force-push
